Manipulando Formularios con JavaScript y el DOM

14 de Diciembre, 2025 10 min de lectura JavaScript DOM Bootstrap

Después de aprender a crear formularios con HTML y Bootstrap, el siguiente paso es darles vida con JavaScript. En este artículo aprenderemos a manipular el DOM para crear un formulario de operaciones matemáticas completamente interactivo.

¿Quieres ver el resultado final? Prueba el formulario interactivo ahora mismo.
Ver Demo

Objetivo del Proyecto

El objetivo es integrar las 4 operaciones matemáticas básicas (suma, resta, multiplicación y división) en un formulario HTML, utilizando JavaScript y el DOM para procesar los datos sin recargar la página.

¿Qué es el DOM? El Document Object Model (DOM) es una interfaz de programación que permite a JavaScript acceder y manipular el contenido, estructura y estilo de un documento HTML.

Estructura HTML del Formulario

Nuestro formulario utiliza Bootstrap 5 y contiene:

  • Dos inputs para ingresar los números
  • Cuatro botones para cada operación matemática
  • Un input deshabilitado para mostrar el resultado

Inputs con Form Floating

Utilizamos el componente Form Floating de Bootstrap para crear inputs con etiquetas flotantes:

<div class="form-floating">
  <input
    class="form-control"
    type="text"
    id="numero1"
    name="numero1"
    placeholder="Numero 1"
    required
  />
  <label for="numero1">Numero 1</label>
  <div class="invalid-feedback">
    Este campo es obligatorio
  </div>
</div>

Características importantes:

  • placeholder es requerido para que funcione el efecto flotante
  • required activa la validación nativa de HTML5
  • invalid-feedback muestra mensajes de error personalizados

Grupo de Botones

Los botones de operaciones están agrupados con btn-group:

<div class="btn-group">
  <button id="sumar" type="submit" class="btn btn-outline-primary">
    Sumar
  </button>
  <button id="restar" type="submit" class="btn btn-outline-primary">
    Restar
  </button>
  <button id="multiplicar" type="submit" class="btn btn-outline-primary">
    Multiplicar
  </button>
  <button id="dividir" type="submit" class="btn btn-outline-primary">
    Dividir
  </button>
</div>

Accediendo al DOM con JavaScript

El primer paso es obtener referencias a los elementos del formulario usando document.getElementById():

// Referencias a los inputs
let form = document.getElementById("form");
let numero1 = document.getElementById("numero1");
let numero2 = document.getElementById("numero2");
let resultado = document.getElementById("resultado");

// Referencias a los botones
let btnSumar = document.getElementById("sumar");
let btnRestar = document.getElementById("restar");
let btnMultiplicar = document.getElementById("multiplicar");
let btnDividir = document.getElementById("dividir");
Tip: Es buena práctica almacenar las referencias en variables para evitar múltiples búsquedas en el DOM, lo cual mejora el rendimiento.

Event Listeners: Escuchando Eventos

Para ejecutar código cuando el usuario hace clic en un botón, utilizamos addEventListener():

btnSumar.addEventListener("click", (e) => {
  // Validar el formulario primero
  if (!validarFormulario(e)) {
    return;
  }

  e.preventDefault(); // Evita que el formulario se envíe

  // Obtener valores y convertirlos a números
  let suma = Number(numero1.value) + Number(numero2.value);

  // Mostrar el resultado
  resultado.value = "El resultado de la suma es: " + suma;
});

Conceptos Clave

Concepto Descripción
addEventListener("click", ...) Escucha el evento de clic en el elemento
e.preventDefault() Evita el comportamiento por defecto (enviar formulario)
Number(valor) Convierte un string a número
input.value Obtiene o establece el valor de un input

Implementando las Operaciones

Cada operación sigue el mismo patrón, con una pequeña variación para la división:

Suma

let suma = Number(numero1.value) + Number(numero2.value);
resultado.value = "El resultado de la suma es: " + suma;

Resta

let resta = Number(numero1.value) - Number(numero2.value);
resultado.value = "El resultado de la resta es: " + resta;

Multiplicación

let multiplicar = Number(numero1.value) * Number(numero2.value);
resultado.value = "El resultado de la multiplicación es: " + multiplicar;

División (con validación especial)

let num1 = Number(numero1.value);
let num2 = Number(numero2.value);

if (num2 === 0) {
  resultado.value = "No se puede dividir números entre 0";
  resultado.classList.add("is-invalid");
} else {
  let dividir = num1 / num2;
  resultado.value = "El resultado de la división es: " + dividir;
}
Importante: La división entre cero es matemáticamente indefinida. Siempre debemos validar que el divisor no sea cero antes de realizar la operación.

Validación del Formulario

Bootstrap 5 incluye estilos de validación que podemos activar con JavaScript:

function validarFormulario(e) {
  let isValid = true;
  resultado.classList.remove("is-invalid");

  if (!form.checkValidity()) {
    event.preventDefault();
    event.stopPropagation();
    isValid = false;
  }

  form.classList.add("was-validated");

  return isValid;
}

¿Cómo funciona?

  1. form.checkValidity() verifica si todos los campos cumplen sus restricciones (como required)
  2. was-validated activa los estilos visuales de validación de Bootstrap
  3. is-invalid marca un campo específico como inválido

Atributos HTML Importantes

Atributo Propósito
novalidate Desactiva la validación nativa del navegador para usar la personalizada
autocomplete="off" Evita que el navegador sugiera valores anteriores
disabled El input de resultado está deshabilitado para evitar edición manual
required Marca el campo como obligatorio

Diseño Responsivo con Grid

El formulario utiliza el sistema de grid de Bootstrap para organizar el contenido:

<div class="row g-4">
  <div class="col-md-7">
    <!-- Formulario -->
  </div>
  <div class="col-md-5">
    <!-- Sidebar con información -->
  </div>
</div>
  • row g-4 crea una fila con gap de 1.5rem
  • col-md-7 ocupa 7 columnas en pantallas medianas y grandes
  • col-md-5 ocupa 5 columnas en pantallas medianas y grandes
  • En móviles, ambas columnas ocupan el 100% del ancho

Conclusión

En este artículo aprendimos a:

  • Acceder a elementos del DOM con getElementById()
  • Escuchar eventos con addEventListener()
  • Prevenir el comportamiento por defecto de formularios
  • Convertir valores de inputs a números con Number()
  • Implementar validación personalizada con Bootstrap
  • Manejar casos especiales como la división entre cero
