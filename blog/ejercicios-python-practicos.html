Ejercicios Prácticos en Python: Palíndromos, Números Primos y Comparación

6 de Enero, 2025 8 min de lectura Python Ejercicios Algoritmos Strings

En este trabajo en clase, resolvimos tres ejercicios prácticos de Python que aplican conceptos fundamentales como condicionales, bucles, manipulación de strings y funciones matemáticas. Aprende a detectar palíndromos, verificar números primos y comparar múltiples valores.

Ejercicios Propuestos

Los tres ejercicios que implementamos en clase fueron:

  1. Ingresar 3 números e indicar cuál es el mayor y el menor
  2. Ingresar un número e indicar si es primo o no
  3. Crear un programa que verifique si una palabra es un palíndromo

Ejercicio 1: Mayor y Menor de Tres Números

Descripción del Problema

Crear un programa que solicite al usuario ingresar tres números enteros y determine cuál es el mayor y cuál es el menor de los tres.

Solución Implementada

num1 = int(input("digite el primer numero: "))
num2 = int(input("digite el segundo numero: "))
num3 = int(input("digite el tercer numero: "))

mayor = max(num1, num2, num3)
menor = min(num1, num2, num3)

print("El numero mayor es: ", mayor)
print("El numero menor es: ", menor)

Análisis del Código

Conceptos Aplicados
  • input() - Solicita datos al usuario
  • int() - Convierte strings a enteros
  • max() - Función integrada que devuelve el valor máximo
  • min() - Función integrada que devuelve el valor mínimo
  • print() - Muestra resultados en consola
Ventaja: Las funciones max() y min() simplifican mucho el código. Sin ellas, necesitaríamos múltiples condicionales if-else para comparar los números.

Ejemplo de Ejecución

digite el primer numero: 45
digite el segundo numero: 12
digite el tercer numero: 78

El numero mayor es:  78
El numero menor es:  12

Ejercicio 2: Verificador de Números Primos

Descripción del Problema

Crear un programa que determine si un número ingresado por el usuario es primo o no. Un número primo es aquel que solo es divisible por 1 y por sí mismo.

Solución Implementada

num1 = int(input("Ingrese el primer numero: "))
esPrimo = True

if num1 <= 1:
    esPrimo = False
for i in range(2, int(num1 ** 0.5) + 1):
    if num1 % i == 0:
        esPrimo = False
        break

if esPrimo:
    print("el numero: ", num1, " es primo")
else:
    print("el numero: ", num1, " no es primo")

Análisis del Algoritmo

Este algoritmo implementa una verificación eficiente de números primos:

  1. Validación inicial: Si el número es menor o igual a 1, no es primo 
    if num1 <= 1:
    esPrimo = False
  2. Optimización con raíz cuadrada: Solo necesitamos verificar divisores hasta la raíz cuadrada del número 
    for i in range(2, int(num1 ** 0.5) + 1):
    ¿Por qué hasta la raíz cuadrada? Si un número tiene un divisor mayor que su raíz cuadrada, también debe tener un divisor menor. Por ejemplo, para 36: si 9 es divisor, entonces 4 también lo es (9 × 4 = 36).
  3. Verificación de divisibilidad: Usamos el operador módulo % 
    if num1 % i == 0:
    esPrimo = False
    break  # Salimos del bucle inmediatamente

Ejemplo de Ejecución

Ejercicio 3: Detector de Palíndromos

Descripción del Problema

Crear un programa que verifique si una palabra ingresada por el usuario es un palíndromo. Un palíndromo es una palabra que se lee igual de izquierda a derecha que de derecha a izquierda.

Solución Implementada

print("Bienvenido a mi aplicación")
letter = input("Ingresa tu palabra:").lower()

if letter == letter[::-1]:
    print("Felicitaciones tu palabra es un palíndromo")
else:
    print("No es palíndromo")

Análisis del Código

1. Normalización del Input

letter = input("Ingresa tu palabra:").lower()

Usamos .lower() para convertir toda la entrada a minúsculas, evitando problemas con mayúsculas/minúsculas. Por ejemplo, "Ana" se convierte en "ana".

2. Inversión de String con Slicing

letter[::-1]

Esta es una técnica muy potente de Python llamada "slice notation":

  • [start:stop:step]
  • [::-1] significa: desde el inicio hasta el final, pero con paso -1 (al revés)
Ejemplos de Slicing
palabra = "python"

palabra[::1]   # "python" (normal)
palabra[::-1]  # "nohtyp" (al revés)
palabra[::2]   # "pto" (cada 2 caracteres)
palabra[1::2]  # "yhn" (desde índice 1, cada 2)
