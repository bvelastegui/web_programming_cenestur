Arreglos en JavaScript: Métodos y Operaciones CRUD

19 de Diciembre, 2025 15 min de lectura JavaScript Arrays CRUD

Los arreglos (arrays) son una de las estructuras de datos más fundamentales en JavaScript. En este artículo aprenderemos a realizar operaciones CRUD, utilizar métodos de iteración como forEach, map y filter, y explorar métodos útiles como join, concat y slice.

¿Qué es un Arreglo?

Un arreglo es una estructura de datos que permite almacenar múltiples valores en una sola variable. En JavaScript, los arreglos pueden contener cualquier tipo de dato: números, strings, objetos, e incluso otros arreglos.

// Arreglo de números
let numeros = [1, 2, 3, 4, 5];

// Arreglo de strings
let frutas = ["Banana", "Apple", "Uva"];

// Longitud del arreglo
console.log(frutas.length); // 3

// Acceder a un elemento por índice (empieza en 0)
console.log(frutas[0]); // "Banana"
Importante: Los índices de los arreglos en JavaScript comienzan en 0, no en 1. El primer elemento está en la posición [0].

Operaciones CRUD

CRUD es un acrónimo que representa las cuatro operaciones básicas que podemos realizar con datos:

Create - Crear
Read - Leer
Update - Actualizar
Delete - Eliminar

Create (Crear)

Para agregar elementos a un arreglo usamos push() y unshift():

let frutas = ["mango", "Apple", "Uva"];

// Agregar al final
frutas.push("Banana");
console.log(frutas); // ["mango", "Apple", "Uva", "Banana"]

// Agregar al principio
frutas.unshift("Pera");
console.log(frutas); // ["Pera", "mango", "Apple", "Uva", "Banana"]

Read (Leer)

Accedemos a los elementos usando su índice:

console.log(frutas);    // Todo el arreglo
console.log(frutas[0]); // Primer elemento: "Pera"
console.log(frutas[2]); // Tercer elemento: "Apple"

Update (Actualizar)

Modificamos elementos asignando un nuevo valor al índice:

frutas[0] = "Mango";
console.log(frutas); // ["Mango", "mango", "Apple", "Uva", "Banana"]

Delete (Eliminar)

Usamos pop(), shift() o splice():

// Eliminar el último elemento
frutas.pop();
console.log(frutas); // ["Mango", "mango", "Apple", "Uva"]

// Eliminar el primer elemento
frutas.shift();
console.log(frutas); // ["mango", "Apple", "Uva"]

// Eliminar en una posición específica (índice 1, eliminar 1 elemento)
frutas.splice(1, 1);
console.log(frutas); // ["mango", "Uva"]

// Agregar en una posición específica (índice 1, eliminar 0, agregar "Kiwi")
frutas.splice(1, 0, "Kiwi");
console.log(frutas); // ["mango", "Kiwi", "Uva"]
Método Acción Posición
push() Agregar Final
unshift() Agregar Inicio
pop() Eliminar Final
shift() Eliminar Inicio
splice() Agregar/Eliminar Cualquiera

Métodos de Iteración

forEach()

Permite iterar sobre cada elemento del arreglo. No modifica el arreglo original y no retorna un nuevo arreglo:

let numeros = [7, 8, 9, 10, 25, 26, 30, 43];

numeros.forEach(numero => {
  console.log(numero);
});
// Imprime: 7, 8, 9, 10, 25, 26, 30, 43

map()

No modifica el arreglo original y retorna un nuevo arreglo con los resultados de aplicar una función a cada elemento:

let numeros = [7, 8, 9, 10, 25, 26, 30, 43];

// Agregar 12% a cada número
let numerosConIVA = numeros.map(numero => numero + (numero * 0.12));

console.log(numerosConIVA);
// [7.84, 8.96, 10.08, 11.2, 28, 29.12, 33.6, 48.16]

console.log(numeros);
// [7, 8, 9, 10, 25, 26, 30, 43] - Sin cambios

filter()

No modifica el arreglo original y retorna un nuevo arreglo con los elementos que cumplan una condición:

let numeros = [7, 8, 9, 10, 25, 26, 30, 43];

// Forma tradicional con for
for (let i = 0; i < numeros.length; i++) {
  if (numeros[i] > 10) console.log(numeros[i]);
}

// Forma moderna con filter
let mayoresA20 = numeros.filter(numero => numero > 20);

console.log(mayoresA20);
// [25, 26, 30, 43]
Diferencia clave: forEach es para ejecutar acciones, map es para transformar datos, y filter es para filtrar datos.

Métodos Útiles Adicionales

join() - Unir elementos en un string

Crea y retorna un nuevo string concatenando todos los elementos del arreglo, separados por un delimitador:

const notas = [10, 8.90, 9.66, 6.5, 9.51, 7.8, 8];

// Unir con separador personalizado
console.log(notas.join(" | "));
// "10 | 8.9 | 9.66 | 6.5 | 9.51 | 7.8 | 8"

// Unir con coma (por defecto)
console.log(notas.join());
// "10,8.9,9.66,6.5,9.51,7.8,8"

concat() - Unir arreglos

Une dos o más arreglos. No modifica los arreglos originales y retorna un nuevo arreglo:

const notas = [10, 8.90, 9.66, 6.5, 9.51, 7.8, 8];
const notasMat = [10, 8, 7.7, 9.2];
const notasDes = [6.5, 8.76, 9.56, 10];

// Unir múltiples arreglos
const todasLasNotas = notas.concat(notasMat, notasDes);

console.log(notas);
// [10, 8.9, 9.66, 6.5, 9.51, 7.8, 8] - Sin cambios

console.log(todasLasNotas);
// [10, 8.9, 9.66, 6.5, 9.51, 7.8, 8, 10, 8, 7.7, 9.2, 6.5, 8.76, 9.56, 10]

slice() - Extraer una porción

Crea una copia superficial de una porción del arreglo. No modifica el arreglo original:

const notas = [10, 8.90, 9.66, 6.5, 9.51, 7.8, 8];
//              0    1     2     3    4     5   6

// Extraer desde índice 3 hasta 5 (no incluye el 5)
const notasMed = notas.slice(3, 5);

console.log(notas);
// [10, 8.9, 9.66, 6.5, 9.51, 7.8, 8] - Sin cambios

console.log(notasMed);
// [6.5, 9.51]
Nota: El segundo parámetro de slice(inicio, fin) indica dónde termina la copia, pero ese elemento no se incluye en el resultado.

sort() - Ordenar elementos

Ordena los elementos del arreglo. Modifica el arreglo original:

const notas = [10, 8.90, 9.66, 6.5, 9.51, 7.8, 8];

// Ordenar de mayor a menor
const notasOrdenadas = notas.sort((a, b) => a < b ? 1 : -1);

// Obtener la nota más alta
const notaMasAlta = notasOrdenadas.at(0);
console.log(notaMasAlta); // 10

Trabajo Individual: CRUD de Calificaciones

Como parte del trabajo individual, se implementó un sistema CRUD para gestionar calificaciones de estudiantes, aplicando los métodos aprendidos:

Requisitos del trabajo
  • Implementar un CRUD de calificaciones
  • Aplicar filter para notas mayores a 7
  • Buscar el promedio más alto
  • Si el estudiante tiene mayor a 9.5, incrementar a 10
  • Investigar y aplicar: join, concat y slice
const notas = [10, 8.90, 9.66, 6.5, 9.51, 7.8, 8];

// CREATE - Agregar notas
notas.push(9.55);      // Al final
notas.unshift(7.77);   // Al principio

// READ - Leer notas
console.log(notas);    // Todo el arreglo
console.log(notas[3]); // Elemento específico

// UPDATE - Actualizar notas mayores a 9.5
for (let i = 0; i < notas.length; i++) {
  if (notas[i] > 9.5) {
    notas[i] = 10;
  }
}

// DELETE - Eliminar notas
notas.pop();           // Última
notas.shift();         // Primera
notas.splice(2, 1);    // Posición específica

// FILTER - Notas mayores a 7
const notasAprobadas = notas.filter(nota => nota > 7);
console.log(notasAprobadas);

// SORT + AT - Encontrar la nota más alta
const notaMasAlta = notas.sort((a, b) => a < b ? 1 : -1).at(0);
console.log(notaMasAlta);
